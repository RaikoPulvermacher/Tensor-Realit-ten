# Pulvermacher – Fundament der Natur  
## Eine Bottom-Up-Beschreibung der Realität

Dieses Repository enthält eine **Bottom-Up-Betrachtung der Natur** nach Raiko Pulvermacher.

Es handelt sich **nicht um ein physikalisches Modell**,  
**nicht um eine formale Theorie**  
und **nicht um eine Konkurrenz** zu bestehenden physikalischen Theorien.

Es ist eine **strukturelle Beschreibung**, wie sich Natur zeigt,  
wenn man sie **nicht trennt**, sondern als **zusammenhängendes Ganzes** betrachtet.

## Kerngedanke

Die Natur existiert **nicht in getrennten Bereichen**.

Energie, Tunnel-Effekt, Information,  
Materie, Gravitation und Zeit  
sind **gleichzeitig vorhanden** und **untrennbar gekoppelt**.

Es gibt keinen bevorzugten Startpunkt.  
Existenz tritt **nur vollständig** auf – oder gar nicht.

---

## Strukturelle Einheiten

**Fundamentale Einheit (quantische Ebene):**

Energie ↔ Tunnel-Effekt ↔ Information  
(E – T – I)

**Makroskopische Entfaltung:**

Zeit ↔ Gravitation ↔ Materie  
(Z – G – M)

**Gesamtstruktur:**

(E, T, I, Z, G, M)

Diese Strukturen sind **keine getrennten Systeme**,  
sondern **zwei Perspektiven derselben Realität**.

---

## Einordnung zu bestehenden Theorien

Top-Down-Theorien  
(z. B. Kopenhagener Deutung, Relativitätstheorie, Standardmodelle)

werden hier **nicht negiert**  
und **nicht abgewertet**.

Sie werden als **Spezialfälle von Messung** verstanden –  
also als Beschreibungen dessen, **was beobachtet wird**,  
nicht dessen, **was ist**.

Dieses Repository beschreibt keine Messvorschrift,  
sondern eine **kohärente Sicht auf Naturzusammenhänge**.
